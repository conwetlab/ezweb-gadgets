Gadgets contributed by CTIC to the EzWeb Platform
-------------------------------------------------

The content of this repository are gadgets under development without any warranty
that can be instantiated from EzWeb. If you want to use any of these gadgets, please
always use the stable versions available at:

    http://idi.fundacionctic.org/ezweb/gadgets

You can find more information in any of the following Web addresses:

    http://www.fundacionctic.org
    http://idi.fundacionctic.org/ezweb
    http://ezweb.morfeo-project.org
    http://ezweb-gadgets.morfeo-project.org
