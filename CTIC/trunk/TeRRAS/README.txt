TeRRAS gadget
=============

TeRRAS Gadget (0.4) for the EzWeb Platform.

More information at: http://terras.sourceforge.net/

Instalation
-----------

Use the guidelines of EzWeb for the installation of new gadgets, taking the
URI of the latest version of this gadget:

    http://idi.fundacionctic.org/ezweb/gadgets/TeRRAS/terras-0.4.xml


Getting the source code
-----------------------

The source code of this gadget is hosted in the MORFEO's forge:

    https://svn.forge.morfeo-project.org/ezweb-gadgets/CTIC/trunk/TeRRAS


Requirements
------------

This gadget requires a valid TeRRAS endpoint. CTIC provides one public endpoint at:

    http://idi.fundacionctic.org/terras/endpoint
